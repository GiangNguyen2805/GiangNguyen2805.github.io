Python Project

While doing this project, I think we faced a little bit of difficulties. But 2 difficulties that stand out the most are when we have to make sure that the word “Ouch!” appeared underneath the road for the hare and the tortoise instead of standing above the path for the race. So what we did was change the location of the if statement for the word “Ouch” to be printed below the road. The second difficulty is when we have to make a full 70 equal signs appear on the road because there are only 69 equal signs and also because everytime we tested the game, either the hare or the tortoise would go even more than 70. So we decided that we would change the phrase from “for i in range (70)” into for i in range (71). Besides that what I did independently was the function for calculating how much the tortoise moves.
